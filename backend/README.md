# Backend - E-Commerce API

## Features
- REST API with CRUD operations
- Product management
- User authentication (basic)
- Order handling

## Tech Stack
- Node.js / ASP.NET Core
- Express / Web API
- SQL Server / MongoDB

## API Endpoints
GET /api/products  
POST /api/products  
PUT /api/products/:id  
DELETE /api/products/:id  

## Run Locally
npm install
npm start
